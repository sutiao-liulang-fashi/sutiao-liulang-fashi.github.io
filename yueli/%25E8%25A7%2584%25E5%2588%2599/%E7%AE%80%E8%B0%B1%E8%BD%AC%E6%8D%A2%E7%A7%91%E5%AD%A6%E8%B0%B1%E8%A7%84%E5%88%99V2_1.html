简谱到科学谱转换规则（基于 ABC 标准 v2.1） ​

概述 ​

简谱是一种使用数字 1-7 表示音高的记谱法，广泛应用于中文音乐教育。科学谱（科学记谱法）是使用音名（A-G）、八度编号和时值修饰符来表示音乐的国际通用方法。本文档详细说明如何将简谱转换为科学谱，以便进一步转换为 ABC 记谱法或其他标准格式。

转换链路：

简谱 → 科学谱 → ABC谱

通过这条转换链路，可以确保三种谱子最终播放和渲染的效果一致。

2. 八度表示 ​

科学谱八度规则 ​

科学谱使用音名后跟数字表示八度：

  • C4 = 中央C（Middle C，MIDI 60）
  • C3 = 低八度（MIDI 48）
  • C5 = 高八度（MIDI 72）

简谱八度表示 ​

  • 无后缀 = 基音八度
  • ’ (单引号) = 升高一个八度
  • '' (双引号) = 升高两个八度
  • , (逗号) = 降低一个八度
  • , (双逗号) = 降低两个八度

计算公式 ​

科学谱八度 = 基音八度 + 简谱八度修饰符

示例（基音C4）：

简谱 1'  → 科学谱 C5 (4 + 1 = 5)
简谱 1,, → 科学谱 C2 (4 - 2 = 2)
简谱 2'  → 科学谱 D5 (4 + 1 = 5)
简谱 5,  → 科学谱 G3 (4 - 1 = 3)

升降号在音名计算中的作用 ​

简谱的升降号会改变音名的计算：

无升降号：简谱 1 → 音名 C
升号：     简谱 1# → 音名 C#

科学谱时值表示 ​

科学谱支持两种时值表示方式：

  1. 整数时值（直接跟在八度后面）

    • C42 = 二分音符（1/4 × 2 = 2/4）
    • C44 = 全音符（1/4 × 4 = 4/4）

  2. 分数时值（使用斜杠）

    • C4/2 = 八分音符（1/4 ÷ 2 = 1/8）
    • C4/4 = 十六分音符（1/4 ÷ 4 = 1/16）

基音处理（关键概念） ​

什么是基音 ​

基音（baseNote）是简谱中数字 1 对应的具体音高。基音的选择会影响整个简谱的音高映射。

G4G大调用于G大调乐曲
F4F大调用于F大调乐曲
A4A大调用于A大调乐曲

转换算法要点 ​

1. 音名计算 ​

简谱数字到音名的映射需要考虑基音和调性：

// 音名数组（按半音排列）
const noteNames = ['C', 'C#', 'D', 'D#', 'E', 'F', 'F#', 'G', 'G#', 'A', 'A#', 'B'];

// 大调音阶偏移
const majorScaleOffset = [0, 2, 4, 5, 7, 9, 11];

// 计算音名
function getNoteName(digit: number, baseNote: string, accidental: string): string {
  // 获取基音的半音位置
  const baseSemitone = getNoteSemitone(baseNote);

  // 计算目标半音位置
  const targetSemitone = (baseSemitone + majorScaleOffset[digit - 1] + getAccidentalOffset(accidental)) % 12;

  // 返回音名
  return noteNames[targetSemitone];
}

3. 时值计算 ​

function calculateDuration(timeModifiers: string[]): string {
  let duration = '';

  // 解析时值修饰符
  for (const modifier of timeModifiers) {
    if (modifier === '-') {
      // 延长线：时值 × 2
      duration = appendMultiplier(duration, 2);
    } else if (modifier === '_') {
      // 下划线：时值 ÷ 2
      duration = appendFraction(duration, 2);
    } else if (modifier === '=') {
      // 双下划线：时值 ÷ 4
      duration = appendFraction(duration, 4);
    } else if (modifier === '.') {
      // 附点：时值 × 1.5
      duration += '.';
    }
  }

  return duration || ''; // 无修饰符返回空
}

function appendMultiplier(current: string, multiplier: number): string {
  if (current === '') {
    return multiplier.toString();
  }
  // 计算新的时值（整数时值）
  const currentValue = parseInt(current, 10) || 1;
  return (currentValue * multiplier).toString();
}

function appendFraction(current: string, divisor: number): string {
  if (current === '') {
    return `/${divisor}`;
  }
  // 计算新的时值（分数时值）
  return `${current}/${divisor}`;
}

注意事项 ​

1. 基音的重要性 ​

  • 基音的选择直接影响简谱到科学谱的音高映射
  • 转换时必须明确指定基音
  • 不同调性应选择对应的基音

2. 八度边界处理 ​

  • 简谱音符7’可能需要调整八度
  • 例如：基音C4，简谱7’ → C5，而不是B4

3. 升降号的影响 ​

  • 简谱的升降号会改变音名计算
  • 例如：简谱1b → B3，而不是C#3

4. 时值修饰符的顺序 ​

  • 时值修饰符按顺序应用
  • 例如：5-. 表示先附点再减半

5. 空格处理 ​

  • 简谱中音符间通常用空格分隔
  • 科学谱中也应保留空格分隔

6. 换行处理 ​

  • 简谱中的小节线可以作为换行提示
  • 科学谱中也应保留小节线

7. 调性一致性 ​

  • 简谱转科学谱时，基音应与调性匹配
  • 例如：G大调应使用G4作为基音

常见错误 ​

  1. ❌ 错误：忽略基音，直接将简谱1转换为C4

    • ✅ 正确：根据调性选择合适的基音

  2. ❌ 错误：简谱7’转换为B4

    • ✅ 正确：简谱7’应转换为C5（跨越八度边界）

  3. ❌ 错误：简谱1b转换为C#4

    • ✅ 正确：简谱1b应转换为B3（降号降低半音）

  4. ❌ 错误：时值修饰符顺序错误

    • ✅ 正确：按从左到右的顺序应用修饰符

  5. ❌ 错误：忽略升降号对音名的影响

    • ✅ 正确：升降号会改变音名计算结果

  6. ❌ 错误：八度计算错误

    • ✅ 正确：仔细计算八度偏移

  7. ❌ 错误：混淆延长线和连音线

    • ✅ 正确：延长线增加时值，连音线连接音符
