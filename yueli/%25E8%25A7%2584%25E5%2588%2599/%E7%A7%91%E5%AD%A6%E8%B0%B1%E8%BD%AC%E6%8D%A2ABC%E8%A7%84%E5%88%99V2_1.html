科学谱到 ABC 记谱法转换规则（基于 ABC 标准 v2.1） ​

概述 ​

科学谱（科学记谱法）是使用音名、八度编号和时值修饰符来表示音乐的一种方法。本文档详细说明如何将科学谱转换为 ABC 记谱法。

基本映射关系 ​

1. 音符名称映射 ​

科学谱使用音名（C, D, E, F, G, A, B）加上八度编号来表示音符。

2. 八度表示（ABC 标准 v2.1） ​

ABC 八度规则 ​

  • CDEFGAB = 第4八度 (C4-B4)
  • cdefghab = 第5八度 (C5-B5)
  • 后缀 ’ (单引号) = 升高一个八度
  • 后缀 , (逗号) = 降低一个八度
  • 支持 , 和 ’ 的组合

科学谱八度到 ABC 映射 ​

示例 ​

科学谱 C3  → ABC C,
科学谱 C4  → ABC C
科学谱 C5  → ABC c
科学谱 C6  → ABC c'
科学谱 C2  → ABC C,,

⚠️ 重要：八度与时值的分离规则 ​

关键规则：科学谱中的第一个数字始终表示八度，后续数字表示时值修饰符。

科学谱八度时值ABC

错误示例：

❌ C42 被错误理解为八度42（会导致大量单引号）
✅ C42 正确理解为八度4 + 时值2

4. 时值表示（ABC 标准 v2.1） ​

基本规则 ​

  • ABC 使用 x/y 格式表示音符长度
  • 默认 x=1, y=2 (四分音符)
  • 在 L:1/4 时：
    • C4 = 四分音符 (1/4)
    • C42 = 二分音符 (2/4)
    • C44 = 全音符 (4/4)
    • C4/2 = 八分音符 (1/8)
    • C4/4 = 十六分音符 (1/16)
    • C4/8 = 三十二分音符 (1/32)

科学谱时值到 ABC 映射 ​

科学谱时值ABC (L:1/4)
C4四分音符C
C42二分音符C2
C44全音符C4
C4/2八分音符C/2
C4/4十六分音符C/4
C4/8三十二分音符C/8

时值格式说明 ​

科学谱支持两种时值表示方式：

  1. 分数时值（使用斜杠）

    • C4/2 = 八分音符（1/4 ÷ 2 = 1/8）
    • C4/4 = 十六分音符（1/4 ÷ 4 = 1/16）
    • C4/8 = 三十二分音符（1/4 ÷ 8 = 1/32）

  2. 整数时值（直接跟在八度后面）

    • C42 = 二分音符（1/4 × 2 = 2/4）
    • C44 = 全音符（1/4 × 4 = 4/4）
    • C48 = 二倍全音符（1/4 × 8 = 8/4）

注意：八度数字和时值数字是相邻的，解析时会根据上下文自动区分：

  • C42：4 是八度，2 是时值（二分音符）
  • C42/2：4 是八度，2 是时值，/2 表示再次减半（二分音符减半 = 四分音符）

5. 附点音符（ABC 标准 v2.1） ​

ABC 附点表示 ​

  • . 在音符后表示附点，音符时值增加一半

示例 ​

科学谱 C4.  → ABC C.    (附点四分音符)
科学谱 C4.2 → ABC C.2   (附点四分音符 + 延长)
科学谱 C4./2 → ABC C./2 (附点八分音符)

附点时值说明 ​

  • C4. = 附点四分音符 (1/4 + 1/8 = 3/8)
  • C4./2 = 附点八分音符 (1/8 + 1/16 = 3/16)

6. 连音线（ABC 标准 v2.1） ​

ABC 连音线表示 ​

  • - 连接两个相同音高的音符（tie），表示它们作为一个音符演奏
  • () 包围一组音符（slur），表示连奏（legato）

科学谱连音线语法 ​

  • 在音符后添加 - 表示连音线

示例 ​

科学谱 C4-C4 (同音连线) → ABC C-C
科学谱 (C4 D4 E4) (连奏) → ABC (C D E)

⚠️ 重要：小节线的使用规则 ​

关键规则：科学谱中的小节线和反复记号不需要与音符用空格分隔，可以连续书写。

错误示例：

❌ "C4 D4 | E4 F4" - 不需要在小节线周围加空格
✅ "C4 D4|E4 F4" - 小节线可以与音符相邻
✅ "C4 D4 | E4 F4" - 也可以加空格（会被正确识别）

转换算法要点 ​

1. 音符解析 ​

格式: [A-G][#b]?\d+[./\d.-]*

2. 八度计算 ​

if octave < 4:
    abcNote = noteName + "," * (4 - octave)
elif octave == 4:
    abcNote = noteName.uppercase()
elif octave == 5:
    abcNote = noteName.lowercase()
else:
    abcNote = noteName.lowercase() + "'" * (octave - 5)

3. 音名转换 ​

将科学谱音名转换为 ABC 格式：
- C → C
- C# → ^C
- Db → _D
- D → D
- D# → ^D
- Eb → _E
- 等等...

4. 时值处理 ​

提取音符后的时值修饰符：
- 无修饰符 → 默认时值
- /2 → /2
- /4 → /4
- 2 → 2
- 4 → 4
- . → .

5. 连音线处理 ​

检测音符后的 - 标记，添加到 ABC 字符串中

注意事项 ​

  1. 大小写敏感: ABC 中 C 和 c 代表不同八度
  2. 空格处理: ABC 中音符间用空格分隔
  3. 换行处理: 每行末尾自动添加换行符
  4. 时值优先级: 附点、延长线、分数时值的顺序很重要
  5. 调性影响: 不同调性下，相同音符可能需要不同的升降号表示
  6. 默认时值: L:1/4 时，无修饰符的音符为四分音符
  7. 连音线: 连音线（-）只能连接相同音高的音符，连奏（()）可以连接不同音高
  8. 多小节休止符: 连续4个或更多休止符可以使用 Z 表示

常见错误 ​

  1. ❌ 错误: C' (C4升八度应该是 c)
  2. ❌ 错误: c, (c5降八度应该是 C)
  3. ❌ 错误: #C (升号应该在前面，应该是 ^C)
  4. ❌ 错误: C/ (斜杠后需要数字，应该是 C/2)
  5. ❌ 错误: C> 后面没有音符（破碎节奏需要两个音符）
  6. ❌ 错误: C4-D4 使用连音线连接不同音高（应该用 () 表示连奏）
  7. ❌ 错误: G42 被误解析为八度42（正确解析：G4八度，2时值，二分音符）
  8. ❌ 错误: C4.2 顺序错误（应该是 C42 或 C4/.2，科学谱中附点通常在时值之后）
  9. ❌ 错误: 混用分数时值和整数时值（如 C42/2 应该改为 C42 或 C4/2）

时值计算公式 ​

  1. 分数时值：时值 = 默认时值 ÷ 分数

    • C4/2 = 1/4 ÷ 2 = 1/8 (八分音符)
    • C4/4 = 1/4 ÷ 4 = 1/16 (十六分音符)

  2. 整数时值：时值 = 默认时值 × 整数

    • C42 = 1/4 × 2 = 2/4 (二分音符)
    • C44 = 1/4 × 4 = 4/4 (全音符)

  3. 附点时值：时值 = 原时值 × 1.5

    • C4. = 1/4 × 1.5 = 3/8 (附点四分音符)
    • C42. = 2/4 × 1.5 = 3/4 (附点二分音符)

优化建议 ​

  1. 性能优化: 使用字符串拼接而非数组join，减少内存分配
  2. 错误处理: 添加对无效科学谱字符的检测和提示
  3. 格式验证: 转换后验证 ABC 格式的正确性
  4. 缓存机制: 对重复的科学谱片段进行缓存
  5. 批量转换: 支持批量转换多个科学谱片段
  6. 多小节休止符: 智能检测连续休止符并使用 Z 表示
  7. 破碎节奏: 优先使用 > 而非 . 表示附点音符对

ABC 标准 v2.1 新特性 ​

  1. 多小节休止符: 支持 Z 和 X 表示多小节休止符
  2. 三连音等: 支持 (2) 到 (9) 的各种连音
  3. 破碎节奏: 支持 > 和 < 以及多重 >> <<<
  4. 虚线小节线: 支持 .| 表示虚线小节线
  5. 不可见小节线: 支持 [|] 表示不可见小节线
  6. 八度组合: 支持 , 和 ' 的组合使用
  7. 附点连音线: 支持 .- 和 .- 表示附点连音线
