# 简谱到科学谱转换规则（基于 ABC 标准 v2.1）

## 概述

简谱是一种使用数字 1-7 表示音高的记谱法，广泛应用于中文音乐教育。科学谱（科学记谱法）是使用音名（A-G）、八度编号和时值修饰符来表示音乐的国际通用方法。本文档详细说明如何将简谱转换为科学谱，以便进一步转换为 ABC 记谱法或其他标准格式。

**转换链路**：
```
简谱 → 科学谱 → ABC谱
```

通过这条转换链路，可以确保三种谱子最终播放和渲染的效果一致。

## 基本映射关系

### 1. 音符名称映射

| 简谱 | 音名 | 科学谱（基音C4） | 说明 |
|------|------|-----------------|------|
| 1 | C | C4 | 基音（中央C） |
| 2 | D | D4 | |
| 3 | E | E4 | |
| 4 | F | F4 | |
| 5 | G | G4 | |
| 6 | A | A4 | |
| 7 | B | B4 | |

**重要**：简谱的音名依赖于基音（baseNote）的选择。基音决定了简谱数字 1 对应的具体音高。

### 2. 八度表示

#### 科学谱八度规则
科学谱使用音名后跟数字表示八度：
- **C4** = 中央C（Middle C，MIDI 60）
- **C3** = 低八度（MIDI 48）
- **C5** = 高八度（MIDI 72）

#### 简谱八度表示
- **无后缀** = 基音八度
- **' (单引号)** = 升高一个八度
- **'' (双引号)** = 升高两个八度
- **, (逗号)** = 降低一个八度
- **,, (双逗号)** = 降低两个八度

#### 简谱到科学谱八度映射（基音C4）

| 简谱 | 相对八度 | 科学谱 | MIDI | 说明 |
|------|---------|--------|------|------|
| 1,, | -2 | C2 | 36 | 低两个八度 |
| 1, | -1 | C3 | 48 | 低一个八度 |
| 1 | 0 | C4 | 60 | 基音（中央C） |
| 1' | +1 | C5 | 72 | 高一个八度 |
| 1'' | +2 | C6 | 84 | 高两个八度 |
| 1''' | +3 | C7 | 96 | 高三个八度 |

#### 计算公式

```
科学谱八度 = 基音八度 + 简谱八度修饰符
```

示例（基音C4）：
```
简谱 1'  → 科学谱 C5 (4 + 1 = 5)
简谱 1,, → 科学谱 C2 (4 - 2 = 2)
简谱 2'  → 科学谱 D5 (4 + 1 = 5)
简谱 5,  → 科学谱 G3 (4 - 1 = 3)
```

#### 不同基音的映射示例

| 基音 | 简谱1 | 简谱1' | 简谱1, |
|------|-------|--------|--------|
| C3 | C3 | C4 | C2 |
| C4 | C4 | C5 | C3 |
| D4 | D4 | D5 | D3 |
| G4 | G4 | G5 | G3 |

### 3. 升降号表示

| 简谱 | 科学谱 | 说明 |
|------|--------|------|
| 1# | C#4 | 升号 (sharp) |
| 1b | B3 | 降号 (flat) |
| 1## | C##4 | 重升号 (double sharp) |
| 1bb | Bbb3 | 重降号 (double flat) |
| 1n | C4 | 还原号 (natural) |

**注意**：简谱中的升降号会影响音名计算。

#### 升降号在音名计算中的作用

简谱的升降号会改变音名的计算：

```
无升降号：简谱 1 → 音名 C
升号：     简谱 1# → 音名 C#
降号：     简谱 1b → 音名 B（因为 Cb = B）
```

### 4. 时值表示

#### 简谱时值表示

| 简谱 | 时值 | 科学谱 | 说明 |
|------|------|--------|------|
| 5 | 四分音符 | C4 | 无修饰符，默认四分音符 |
| 5- | 二分音符 | C42 | 延长线，时值×2 |
| 5-- | 全音符 | C44 | 双延长线，时值×4 |
| 5_ | 八分音符 | C4/2 | 下划线，时值÷2 |
| 5= | 十六分音符 | C4/4 | 双下划线，时值÷4 |
| 5. | 附点四分音符 | C4. | 附点，时值×1.5 |

#### 科学谱时值表示

科学谱支持两种时值表示方式：

1. **整数时值**（直接跟在八度后面）
   - `C42` = 二分音符（1/4 × 2 = 2/4）
   - `C44` = 全音符（1/4 × 4 = 4/4）

2. **分数时值**（使用斜杠）
   - `C4/2` = 八分音符（1/4 ÷ 2 = 1/8）
   - `C4/4` = 十六分音符（1/4 ÷ 4 = 1/16）

#### 简谱到科学谱时值映射

| 简谱 | 时值 | 科学谱（推荐格式） | 时值计算 |
|------|------|-------------------|---------|
| 5 | 1/4 | C4 | 默认 |
| 5- | 2/4 | C42 | 1/4 × 2 |
| 5-- | 4/4 | C44 | 1/4 × 4 |
| 5_ | 1/8 | C4/2 | 1/4 ÷ 2 |
| 5= | 1/16 | C4/4 | 1/4 ÷ 4 |
| 5=== | 1/32 | C4/8 | 1/4 ÷ 8 |
| 5. | 3/8 | C4. | 1/4 × 1.5 |
| 5-. | 3/16 | C4./2 | (1/4 × 1.5) ÷ 2 |

**推荐格式**：
- 使用整数时值表示延长线（如 C42）
- 使用分数时值表示减半（如 C4/2）
- 使用附点表示附点音符（如 C4.）

### 5. 附点音符

#### 简谱附点表示
- **.** 在音符后表示附点，音符时值增加一半

#### 科学谱附点表示
- **.** 在音符后表示附点，与简谱相同

#### 示例

| 简谱 | 科学谱 | 时值 | 说明 |
|------|--------|------|------|
| 5. | C4. | 3/8 | 附点四分音符 |
| 5-. | C4./2 | 3/16 | 附点八分音符 |
| 5.- | C42. | 3/4 | 附点二分音符 |
| 5--. | C44. | 3/2 | 附点全音符 |

### 6. 连音线

#### 简谱连音线表示
- **-** 连接两个相同音高的音符（tie）
- **()** 包围一组音符（slur/legato）

#### 科学谱连音线表示
- **-** 在音符后表示连音线（tie）
- **()** 包围一组音符（slur/legato）

#### 示例

| 简谱 | 科学谱 | 说明 |
|------|--------|------|
| 5-5 | C4-C4 | 同音连线，作为一个音符演奏 |
| (5 6 7) | (C4 D4 E4) | 连奏，平滑连接 |
| 5-6-7 | (C4 D4 E4) | 连奏的另一种表示 |

### 7. 小节线和反复记号

| 简谱 | 科学谱 | 说明 |
|------|--------|------|
| \| | \| | 小节线 |
| \|\| | \|\| | 双小节线 |
| \|: | \|: | 反复开始 |
| :\| | :\| | 反复结束 |
| :: | :: | 双反复（开始+结束） |
| \|:: | \|:: | 三次反复开始 |
| ::\| | ::\| | 三次反复结束 |

#### 示例

```
简谱 |: 1 2 3 1 :|
科学谱 |: C4 D4 E4 C4 :|

简谱 1 2 | 3 4 || 5 6
科学谱 C4 D4 | E4 F4 || G4 A4
```

### 8. 休止符

| 简谱 | 科学谱 | 说明 |
|------|--------|------|
| - 或 0 | z | 休止符（可见） |
| Z4 | Z4 | 多小节休止符（4小节） |
| Z | Z | 一小节休止符 |

#### 示例

```
简谱 5 0 3 0 | Z4 |
科学谱 G4 z E4 z | Z4 |
```

### 9. 三连音等特殊记号

| 记号 | 简谱 | 科学谱 | 说明 |
|------|------|--------|------|
| 三连音 | (1 2 3) | (3 C4 D4 E4) | 3个音符在2个音符的时间内 |
| 二连音 | (1 2) | (2 C4 D4) | 2个音符在3个音符的时间内 |
| 四连音 | (1 2 3 4) | (4 C4 D4 E4 F4) | 4个音符在3个音符的时间内 |
| 五连音 | (1 2 3 4 5) | (5 C4 D4 E4 F4 G4) | 5个音符在n个音符的时间内 |

## 基音处理（关键概念）

### 什么是基音

基音（baseNote）是简谱中数字 1 对应的具体音高。基音的选择会影响整个简谱的音高映射。

### 常见基音选择

| 基音 | 适用调性 | 说明 |
|------|---------|------|
| C4 | C大调 | 最常用，中央C |
| D4 | D大调 | 用于D大调乐曲 |
| G4 | G大调 | 用于G大调乐曲 |
| F4 | F大调 | 用于F大调乐曲 |
| A4 | A大调 | 用于A大调乐曲 |

### 基音对转换的影响

示例：简谱 "1 2 3 4" 在不同基音下的科学谱表示：

| 基音 | 简谱1 | 简谱2 | 简谱3 | 简谱4 |
|------|-------|-------|-------|-------|
| C4 | C4 | D4 | E4 | F4 |
| D4 | D4 | E4 | F#4 | G4 |
| G4 | G4 | A4 | B4 | c5 |
| F4 | F4 | G4 | A4 | Bb4 |

### 计算基音的八度

```
基音八度 = parseInt(baseNote.match(/\d+$/)[0], 10)
```

示例：
- C4 → 基音八度 = 4
- G4 → 基音八度 = 4
- c5 → 基音八度 = 5

## 转换算法要点

### 1. 音名计算

简谱数字到音名的映射需要考虑基音和调性：

```
// 音名数组（按半音排列）
const noteNames = ['C', 'C#', 'D', 'D#', 'E', 'F', 'F#', 'G', 'G#', 'A', 'A#', 'B'];

// 大调音阶偏移
const majorScaleOffset = [0, 2, 4, 5, 7, 9, 11];

// 计算音名
function getNoteName(digit: number, baseNote: string, accidental: string): string {
  // 获取基音的半音位置
  const baseSemitone = getNoteSemitone(baseNote);

  // 计算目标半音位置
  const targetSemitone = (baseSemitone + majorScaleOffset[digit - 1] + getAccidentalOffset(accidental)) % 12;

  // 返回音名
  return noteNames[targetSemitone];
}
```

#### 音名计算示例

**基音 C4，无升降号**：
```
简谱 1 → C4 (0 + 0 = 0 → C)
简谱 2 → D4 (0 + 2 = 2 → D)
简谱 3 → E4 (0 + 4 = 4 → E)
简谱 4 → F4 (0 + 5 = 5 → F)
简谱 5 → G4 (0 + 7 = 7 → G)
简谱 6 → A4 (0 + 9 = 9 → A)
简谱 7 → B4 (0 + 11 = 11 → B)
```

**基音 C4，有升降号**：
```
简谱 1# → C#4 (0 + 0 + 1 = 1 → C#)
简谱 2# → D#4 (0 + 2 + 1 = 3 → D#)
简谱 4# → F#4 (0 + 5 + 1 = 6 → F#)
简谱 5# → G#4 (0 + 7 + 1 = 8 → G#)
简谱 7# → B#4 (0 + 11 + 1 = 0 → C)  // B# = C
```

```
function calculateOctave(digit: number, octaveModifier: number, baseOctave: number): number {
  // 默认八度是基音八度
  let octave = baseOctave + octaveModifier;

  // 特殊处理：音符7升高后可能需要增加八度
  if (digit === 7 && octaveModifier === 0) {
    // 检查是否跨越八度边界
    // 例如：基音C4，简谱7 → B4，不需要调整
    // 但如果有升降号，可能需要调整
  }

  return octave;
}
```

#### 八度计算示例

**基音 C4**：
```
简谱 1,, → C2 (4 - 2 = 2)
简谱 1,  → C3 (4 - 1 = 3)
简谱 1   → C4 (4 + 0 = 4)
简谱 1'  → C5 (4 + 1 = 5)
简谱 1'' → C6 (4 + 2 = 6)
```

**基音 G4**：
```
简谱 1,, → G2 (4 - 2 = 2)
简谱 1,  → G3 (4 - 1 = 3)
简谱 1   → G4 (4 + 0 = 4)
简谱 1'  → G5 (4 + 1 = 5)
简谱 1'' → G6 (4 + 2 = 6)
```

### 3. 时值计算

```
function calculateDuration(timeModifiers: string[]): string {
  let duration = '';

  // 解析时值修饰符
  for (const modifier of timeModifiers) {
    if (modifier === '-') {
      // 延长线：时值 × 2
      duration = appendMultiplier(duration, 2);
    } else if (modifier === '_') {
      // 下划线：时值 ÷ 2
      duration = appendFraction(duration, 2);
    } else if (modifier === '=') {
      // 双下划线：时值 ÷ 4
      duration = appendFraction(duration, 4);
    } else if (modifier === '.') {
      // 附点：时值 × 1.5
      duration += '.';
    }
  }

  return duration || ''; // 无修饰符返回空
}

function appendMultiplier(current: string, multiplier: number): string {
  if (current === '') {
    return multiplier.toString();
  }
  // 计算新的时值（整数时值）
  const currentValue = parseInt(current, 10) || 1;
  return (currentValue * multiplier).toString();
}

function appendFraction(current: string, divisor: number): string {
  if (current === '') {
    return `/${divisor}`;
  }
  // 计算新的时值（分数时值）
  return `${current}/${divisor}`;
}
```

#### 时值计算示例

| 简谱 | 修饰符 | 科学谱时值 | 计算过程 |
|------|--------|-----------|---------|
| 5 | 无 | （无） | 默认四分音符 |
| 5- | - | 2 | 1 × 2 = 2 |
| 5-- | -,- | 4 | 1 × 2 × 2 = 4 |
| 5_ | _ | /2 | 默认 ÷ 2 |
| 5=_ | _,= | /4 | 默认 ÷ 2 ÷ 4 |
| 5. | . | . | 默认 × 1.5 |
| 5-. | -,. | ./2 | (默认 × 1.5) ÷ 2 |
| 5.- | .,- | 2. | (默认 × 1.5) × 2 |

### 三种谱子的等价性验证

通过以下转换链路验证三种谱子的等价性：

```
简谱 → 科学谱 → ABC谱
```

#### 验证示例1

```
1 2 3 4 | 5 6 7 1' |
```

**科学谱（中间结果）：**
```

```

```
C D E F | G A B c |
```

**验证结果**：✅ 三种谱子播放效果一致

**科学谱（中间结果）：**
```

```

```
G. E G. E | C D E C |
```

**验证结果**：✅ 三种谱子播放效果一致

#### 验证示例3（不同基音）

```
1 2 3 4 | 5 6 7 1' |
```

**科学谱（中间结果）：**
```

```

## 注意事项

### 1. 基音的重要性
- 基音的选择直接影响简谱到科学谱的音高映射
- 转换时必须明确指定基音
- 不同调性应选择对应的基音

### 2. 八度边界处理
- 简谱音符7'可能需要调整八度
- 例如：基音C4，简谱7' → C5，而不是B4

### 3. 升降号的影响
- 简谱的升降号会改变音名计算
- 例如：简谱1b → B3，而不是C#3

### 4. 时值修饰符的顺序
- 时值修饰符按顺序应用
- 例如：5-. 表示先附点再减半

### 5. 空格处理
- 简谱中音符间通常用空格分隔
- 科学谱中也应保留空格分隔

### 6. 换行处理
- 简谱中的小节线可以作为换行提示
- 科学谱中也应保留小节线

### 7. 调性一致性
- 简谱转科学谱时，基音应与调性匹配
- 例如：G大调应使用G4作为基音

## 常见错误

1. ❌ **错误**：忽略基音，直接将简谱1转换为C4
   - ✅ **正确**：根据调性选择合适的基音

2. ❌ **错误**：简谱7'转换为B4
   - ✅ **正确**：简谱7'应转换为C5（跨越八度边界）

3. ❌ **错误**：简谱1b转换为C#4
   - ✅ **正确**：简谱1b应转换为B3（降号降低半音）

4. ❌ **错误**：时值修饰符顺序错误
   - ✅ **正确**：按从左到右的顺序应用修饰符

5. ❌ **错误**：忽略升降号对音名的影响
   - ✅ **正确**：升降号会改变音名计算结果

6. ❌ **错误**：八度计算错误
   - ✅ **正确**：仔细计算八度偏移

7. ❌ **错误**：混淆延长线和连音线
   - ✅ **正确**：延长线增加时值，连音线连接音符
