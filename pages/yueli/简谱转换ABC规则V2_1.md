# 简谱到 ABC 记谱法转换规则（基于 ABC 标准 v2.1）

## 基本映射关系

### 1. 音符名称映射

| 简谱 | 音名 | ABC |
|------|------|-----|
| 1 | C | C |
| 2 | D | D |
| 3 | E | E |
| 4 | F | F |
| 5 | G | G |
| 6 | A | A |
| 7 | B | B |

### 2. 八度表示（ABC 标准 v2.1）

#### ABC 八度规则
- **CDEFGAB** = 第4八度 (C4-B4)
- **cdefghab** = 第5八度 (C5-B5)
- 后缀 **'** (单引号) = 升高一个八度
- 后缀 **,** (逗号) = 降低一个八度
- 支持 **,** 和 **'** 的组合，如 `C,'` 等效于 `C`

#### 示例
```
简谱 1   → ABC C     (C4)
简谱 1'  → ABC c     (C5)
简谱 1'' → ABC c'    (C6)
简谱 1,  → ABC C,    (C3)
简谱 1,, → ABC C,,   (C2)
```

### 3. 升降号表示（ABC 标准 v2.1）

| 简谱 | ABC | 说明 |
|------|-----|------|
| 1# | ^C | 升号 (sharp) |
| 1b | _C | 降号 (flat) |
| 1## | ^^C | 重升号 (double sharp) |
| 1bb | __C | 重降号 (double flat) |
| 1n | =C | 还原号 (natural) |

### 4. 时值表示（ABC 标准 v2.1）

#### 基本规则
- ABC 使用 **x/y** 格式表示音符长度
- 默认 **x=1, y=2** (四分音符)
- 在 **L:1/4** 时：
  - **C** 或 **C1** = 四分音符 (1/4)
  - **C2** = 二分音符 (2/4)
  - **C3** = 附点四分音符 (3/4)
  - **C4** = 全音符 (4/4)
  - **C6** = 附点二分音符 (6/4)
  - **C/2** = 八分音符 (1/8)
  - **C/4** = 十六分音符 (1/16)
  - **C/8** = 三十二分音符 (1/32)

#### 简谱时值到 ABC 映射

| 简谱 | 时值 | ABC (L:1/4) |
|------|------|-------------|
| 5 | 四分音符 | 5 |
| 5- | 二分音符 | 52 |
| 5-- | 全音符 | 54 |
| 5_ | 八分音符 | 5/2 |
| 5= | 十六分音符 | 5/4 |
| 5. | 附点四分音符 | 5. |

### 5. 附点音符（ABC 标准 v2.1）

#### ABC 附点表示
- **.** 在音符后表示附点，音符时值增加一半
- **>** 表示破碎节奏（broken rhythm），左边的音延长一半，右边的音缩短一半

#### 示例
```
简谱 5.  → ABC 5.    (附点四分音符)
简谱 5.3 → ABC 5.3   (附点四分音符 + 四分音符)
简谱 5>3 → ABC 5>3   (附点四分音符 + 八分音符，破碎节奏)
```

### 6. 连音线（ABC 标准 v2.1）

#### ABC 连音线表示
- **-** 连接两个相同音高的音符（tie），表示它们作为一个音符演奏
- **()** 包围一组音符（slur），表示连奏（legato）

#### 示例
```
简谱 5-5 (同音连线) → ABC 5-5
简谱 (5 6 7) (连奏) → ABC (5 6 7)
```

### 7. 小节线和反复记号（ABC 标准 v2.1）

| 符号 | ABC | 说明 |
|------|-----|------|
| \| | \| | 小节线 |
| \|\| | \|\| | 双小节线 |
| \|] | \|] | 细-粗双小节线 |
| [\| | [\| | 粗-细双小节线 |
| \|: | \|: | 反复开始 |
| :\| | :\| | 反复结束 |
| :: | :: | 双反复（开始+结束） |
| \|:: | \|:: | 三次反复开始 |
| ::\| | ::\| | 三次反复结束 |
| .\| | .\| | 虚线小节线 |
| [\|] | [\|] | 不可见小节线 |

### 8. 休止符（ABC 标准 v2.1）

| 简谱 | ABC | 说明 |
|------|-----|------|
| - 或 0 | z | 休止符（可见） |
| - 或 0 | x | 休止符（不可见） |
| Z4 | Z4 | 多小节休止符（4小节） |
| Z | Z | 一小节休止符 |
| X4 | X4 | 多小节不可见休止符（4小节） |

### 9. 三连音等特殊记号（ABC 标准 v2.1）

| 记号 | ABC | 说明 |
|------|-----|------|
| 三连音 | (3abc | 3个音符在2个音符的时间内 |
| 二连音 | (2ab | 2个音符在3个音符的时间内 |
| 四连音 | (4abcd | 4个音符在3个音符的时间内 |
| 五连音 | (5abcde | 5个音符在n个音符的时间内 |
| 六连音 | (6abcdef | 6个音符在2个音符的时间内 |

## 转换算法要点

### 1. 基音处理
- 简谱的 "1" 对应基音（如 C4）
- 根据基音计算其他音符的音高

### 2. 八度计算
```
finalOctave = baseOctave + octaveModifier
```

### 3. 音名计算
```
semitone = (baseSemitone + scaleOffset[digit] + accidental) % 12
noteName = noteNames[semitone]
```

### 4. 时值计算
```
duration = baseDuration * timeModifier
```

### 5. ABC 八度格式转换
```
if octave < baseOctave:
    abcNote = noteName + "," * (baseOctave - octave)
elif octave == baseOctave:
    abcNote = noteName.uppercase()
elif octave == baseOctave + 1:
    abcNote = noteName.lowercase()
else:
    abcNote = noteName.lowercase() + "'" * (octave - baseOctave - 1)
```

## 注意事项

1. **大小写敏感**: ABC 中 C 和 c 代表不同八度
2. **空格处理**: ABC 中音符间用空格分隔
3. **换行处理**: 每行末尾自动添加换行符
4. **时值优先级**: 附点、延长线、减半等修饰符的顺序很重要
5. **调性影响**: 不同调性下，相同简谱数字对应不同音名
6. **默认时值**: L:1/4 时，无修饰符的音符为四分音符
7. **连音线**: 连音线（-）只能连接相同音高的音符，连奏（()）可以连接不同音高
8. **多小节休止符**: 连续4个或更多休止符可以使用 Z 表示

## 常见错误

1. ❌ 错误: `C'` (C4升八度应该是 `c`)
2. ❌ 错误: `c,` (c5降八度应该是 `C`)
3. ❌ 错误: `#C` (升号应该在前面，应该是 `^C`)
4. ❌ 错误: `C/` (斜杠后需要数字，应该是 `C/2`)
5. ❌ 错误: `C>` 后面没有音符（破碎节奏需要两个音符）
6. ❌ 错误: `5-6` 使用连音线连接不同音高（应该用 `()` 表示连奏）

## 优化建议

1. **性能优化**: 使用字符串拼接而非数组join，减少内存分配
2. **错误处理**: 添加对无效简谱字符的检测和提示
3. **格式验证**: 转换后验证 ABC 格式的正确性
4. **缓存机制**: 对重复的简谱片段进行缓存
5. **批量转换**: 支持批量转换多个简谱片段
6. **多小节休止符**: 智能检测连续休止符并使用 Z 表示
7. **破碎节奏**: 优先使用 `>` 而非 `. ` 表示附点音符对

## ABC 标准 v2.1 新特性

1. **多小节休止符**: 支持 `Z` 和 `X` 表示多小节休止符
2. **三连音等**: 支持 `(2)` 到 `(9)` 的各种连音
3. **破碎节奏**: 支持 `>` 和 `<` 以及多重 `>>` `<<<`
4. **虚线小节线**: 支持 `.|` 表示虚线小节线
5. **不可见小节线**: 支持 `[|]` 表示不可见小节线
6. **八度组合**: 支持 `,` 和 `'` 的组合使用
7. **附点连音线**: 支持 `.-` 和 `.-` 表示附点连音线
